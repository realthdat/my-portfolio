## 🖥️ Getting Started

Clone the repository and install dependencies:

```bash
git clone https://github.com/realthdat/my-portfolio.git
cd my-portfolio
npm install
# or
yarn install
```

Start the development server:

```bash
npm run dev
# or
yarn dev
```

Visit [http://localhost:3000](http://localhost:3000) in your browser.
